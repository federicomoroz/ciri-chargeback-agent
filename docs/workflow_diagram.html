Decisiones de diseño
  • n8n orquesta, FastAPI ejecuta
  • QueryBuilder determinístico, sin LLM para queries
  • Políticas editables vía API, sin redeploy
  • Guardrail: APPROVE + BLOCKER → hallucination
  • Caché semántico, threshold 0.92
  • Prompts versionados (v1_*)
RAG pipeline
  • Embeddings: Voyage AI (voyage-multilingual-2)
  • 1024 dims, multilingual, Voyage AI API
  • Políticas: top-17, threshold 0.0
  • Casos similares: top-5, threshold 0.40
  • SQLite + Qdrant unidos en contexto LLM
  • Auto-indexing: score ≥ 8.0 → nuevo precedente
Deploy + Observabilidad
  • API: Render.com, Docker
  • Qdrant Cloud, sa-east-1
  • n8n Cloud, 42 nodos
  • LLM: Claude Haiku 4.5 (Anthropic)
  • Langfuse: tokens, latencia, judge scores
  • 203 tests, pytest, mocks LLM
